# Simple RPP Tracker

This is a personal project with two main objectives:

1. Produce a working russian fighter pull up program tracker, since I cannot find one anywhere on the internet.
2. Test the capability of claude code, without me manually typing or editing a single line of code myself (repo starts COMPLETELY empty, no initial files or folders, nor claude.md - all of it has to be done by Claude Code). The only file I will manually type into is this readme.md.

Features may be added in the future as I get more ideas for the application. It may be expanded to have a backend + database if the need arises.

If you are interested in using the tracker: https://r-lok.github.io/simple-rpp-tracker/
